🎉 EventFlow – Event Management System

A simple and responsive web application to create, edit, and manage events efficiently. This project helps users organize events, track schedules, and maintain event details using an intuitive interface.

🚀 Features

📅 Create new events

✏️ Edit existing events

🗑️ Delete events

📋 View all scheduled events

🧭 Calendar-based event navigation

⚡ Fast and responsive UI

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript

Storage: Browser LocalStorage

Tools: VS Code, Live Server
